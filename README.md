#  Terminal NIRD - Le Village Numérique Résistant

> Projet réalisé lors de la **Nuit de l'Info 2025** par l'équipe **Le Genou de Tony**.

(https://via.placeholder.com/800x400.png?text=Terminal+NIRD+-+Le+Genou+de+Tony)

##  Présentation du Sujet

[cite_start]Ce projet répond au défi **"Le Village Numérique Résistant : Comment les établissements scolaires peuvent tenir tête aux Big Tech ?"**[cite: 8].

[cite_start]Face à la fin du support de Windows 10 et à l'hégémonie des GAFAM [cite: 10][cite_start], notre application propose une approche ludique pour sensibiliser à la démarche **NIRD** (Numérique Inclusif, Responsable et Durable)[cite: 12].

[cite_start]L'objectif : Prouver qu'une école peut devenir un village résistant, ingénieux et autonome[cite: 11].

##  Le Concept

Nous avons pris le parti d'une **interface immersive "Hacker / Terminal"**. L'utilisateur incarne l'administrateur système du Lycée Carnot et doit naviguer dans l'arborescence des fichiers pour résoudre des crises numériques.

### Fonctionnalités principales :
* **Simulation CLI (Command Line Interface)** : Navigation réaliste via une arborescence de fichiers (`tree`).
* **Scénarios à choix multiples** : Chaque décision impacte trois jauges vitales :
    *  **Liberté** (Indépendance technologique)
    *  **Écologie** (Durabilité du matériel)
    *  **Budget** (Coût des licences)
* [cite_start]**Pédagogie** : Promotion des logiciels libres (Linux, Nextcloud) et du réemploi[cite: 52, 53].
* **Ambiance Rétro** : Effets sonores visuels (Scanlines CRT, typographie Monospace, code couleur Matrix).

##  Easter Egg (Bonus Caché)

Comme demandé par le défi, un jeu secret est caché dans l'application.

**Pour activer le Snake 3D Holographique :**
Effectuez le **Konami Code** sur votre clavier n'importe où sur la page :
`↑` `↑` `↓` `↓` `←` `→` `←` `→` `B` `A`

*Caractéristiques du Snake :*
* Moteur de jeu codé en Vanilla JS.
* Rendu 3D isométrique via Canvas et transformations CSS.
* Score dynamique.

##  Installation et Lancement

Ce projet est conçu pour être ultra-léger et ne nécessite aucune compilation.

1.  **Cloner le dépôt :**
    ```bash
    git clone [https://github.com/NoNoyan02/nuit-de-l-info-2025.git)
    ```
2.  **Lancer le projet :**
    * Ouvrez simplement le fichier `index.html` dans n'importe quel navigateur web moderne (Chrome, Firefox, Edge).
    * Ou hébergez-le sur GitHub Pages / Vercel / Netlify en un clic.

##  Technologies Utilisées

* **HTML5** : Structure sémantique.
* **CSS3** : Animations, Flexbox, Grid, Effets CRT et 3D Transforms.
* **JavaScript (Vanilla)** : Logique du jeu, gestion du DOM, Canvas (pour le Snake).
* **Aucun Framework** : Performance maximale et empreinte écologique minimale.
